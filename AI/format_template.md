# ИНСТРУКЦИЯ: ФОРМАТИРОВАНИЕ HTML И PYTHON КОДА

## ЗАДАЧА
На вход получаешь HTML-шаблон с вставками Python 2.7 кода в фигурных скобках `{python_code}`.
Необходимо отформатировать отступы для читаемости:
1. Весь HTML код вне фигурных скобок
2. Python 2.7 код внутри фигурных скобок
3. **ДОБАВЛЯТЬ ОТСТУП для Python блоков, если они находятся внутри HTML тегов**

## ЖЕСТКИЕ ЗАПРЕТЫ - НИКАКИХ ИЗМЕНЕНИЙ!
- ЗАПРЕЩЕНО менять логику Python кода
- ЗАПРЕЩЕНО менять HTML структуру (теги, атрибуты, порядок)
- ЗАПРЕЩЕНО предлагать улучшения, оптимизации, исправления
- ЗАПРЕЩЕНО менять синтаксис Python 2.7 на Python 3
- ЗАПРЕЩЕНО удалять или добавлять функциональность
- ЗАПРЕЩЕНО менять содержимое текстовых узлов HTML
- ЗАПРЕЩЕНО переименовывать теги или атрибуты

## АЛГОРИТМ ОБРАБОТКИ

### ШАГ 1: ВЫДЕЛИ PYTHON БЛОКИ
1. Найди все участки между `{` и `}`
2. Проверь, содержит ли блок код Python 2.7
3. Запомни точные позиции и содержимое этих блоков
4. Определи, внутри каких HTML тегов находится каждый Python блок

### ШАГ 2: ОТФОРМАТИРУЙ HTML (между Python блоками)
Для HTML частей ВНЕ фигурных скобок:
- Теги `<html>`, `<head>`, `<body>` должны быть **БЕЗ ОТСТУПА** (0 пробелов) 
- Используй отступ 2 пробела для вложенности внутри тегов `<head>`, `<body>` 
- Каждый вложенный тег увеличивай отступ на 2 пробела
- Закрывающие теги выравнивай с открывающими
- Пустые теги оставляй как есть: `<div></div>` или `<div />`
- Самозакрывающиеся теги: `<br />`, `<img />`, `<input />`
- Не меняй HTML-сущности: `&nbsp;`, `&lt;`, `&gt;`, `&amp;`

### ШАГ 3: ОТФОРМАТИРУЙ PYTHON КОД (внутри блоков)
Для каждого Python блока внутри `{ }`:

1. **ОПРЕДЕЛИ УРОВЕНЬ ВЛОЖЕННОСТИ В HTML:**
   - Если Python блок находится внутри открытого HTML тега, добавь отступ 4 пробела
   - Если вложен в несколько тегов, добавь отступ 2 пробела на каждый уровень вложенности
   - Пример: Python блок внутри `<div><span>` должен иметь 8 пробелов отступа

2. **ОТФОРМАТИРУЙ САМ PYTHON КОД:**
   - Используй отступ 2 пробела для Python кода (дополнительно к HTML отступу)
   - Вложенные Python блоки: +2 пробела на каждый уровень
   - Сохрани синтаксис Python 2.7:
     - `print "text"` (без скобок)
     - `except Exception, e:` (старый синтаксис)
     - `u"unicode текст"`
     - Целочисленное деление: `5 / 2 = 2`
   - Не исправляй синтаксические ошибки в Python коде!

3. **РАССТАНОВКА ПЕРЕНОСОВ СТРОК для Python блоков внутри тегов:**
   - Каждый Python блок внутри HTML тега начинай с новой строки
   - Добавляй отступ в соответствии с уровнем вложенности HTML тега
   - Сохраняй пустые строки между логическими блоками Python кода для читаемости

### ШАГ 4: ОСОБОЕ ПРАВИЛО - ВЛОЖЕННЫЕ PYTHON БЛОКИ ВНУТРИ HTML
Если Python блоки логически связаны и находятся внутри одного HTML тега:
1. Выровняй их по одному уровню отступа
2. Добавь отступ для вложенных Python конструкций
3. Пример преобразования:

**БЫЛО:**
```html
<div>{: test = 0} {if: len(arr) >0} {:test = 1}{end:}</div>
```

**СТАЛО:**
```html
<div>
  {: test = 0} 
  {if: len(arr) >0} 
    {:test = 1}
  {end:}
</div>
```
