## FILE NAMING RULE
- имя файла всегда вычисляется, а не придумывается
- запрещено использовать произвольные названия
- запрещено добавлять префиксы/суффиксы вне правила
Имя нового файла должно формироваться по следующему правилу:

- формат: <template_name>_<index>.html

Где:
- template_name — имя шаблона, извлечённое из PDF/DOCX (или логическое название секции макета)
- index — порядковый номер файла, начиная с 1

## RULES FOR template_name:
- берётся из названия секции в макете (PDF/DOCX)
- приводится к lowercase
- пробелы заменяются на _
- не содержит спецсимволов

## INDEX RULES:
- index = 1 для первого файла данного template_name
- если файл уже существует в ai/templates:
  - index увеличивается на +1
- агент обязан проверять существующие файлы перед созданием нового
